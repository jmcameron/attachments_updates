Attachments 3.1.3 for Joomla 2.5+ and 3.x

Joomla 3.1 Compatiblity Release

November 15, 2013

Release Notes

  • Fixed issue with displaying options with Joomla 3.2
  • Update to allow display of attachments lists in editors for non-article.
  • Fixed migration importer handle obsolete 'section' attachments
  • Fixed bug in 'Show attachments to non logged-in users' option
  • NOTE: To uninstall the Attachments extension properly, uninstall the 'Attachments Package' which will uninstall the Attachments component and all related plugins.
