Attachments 3.2.4 for Joomla 2.5+ and 3.x

May 7, 2016

Release Notes

  • Updated to work with Joomla 3.5.1
  • Fixed admin [Reset Order] button
  • Fixed issue with uploads in Joomla 3.4.4
  • Fixed showing attachments in TinyMCE editor
  • Fixed modal issues with TinyMCE editor on J3.5.1
  • Tested on Joomla 3.4.8 and 3.5.1

Download: attachments-3.2.4.zip

NOTE: To uninstall the Attachments extension properly, uninstall the 'Attachments Package' which will uninstall the Attachments component and all related plugins.
